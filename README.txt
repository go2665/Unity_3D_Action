간단 Unity 3D 액션 게임
유니티 6000.1.6f1

조작법
 - 캐릭터 조작
    - WASD : 이동
    - 마우스 왼쪽 클릭 : 공격
    - 마우스 오른쪽 클릭 : 스킬
    - E : 아이템 획득
    - F : 적 락온
    - I : 인벤토리 열기
 - 인벤토리 조작
    - 드래그 : 아이템 이동
    - 클릭 : 아이템 장비 or 아이템 사용
    - Shift+클릭 : 아이템 나누기

학습 주제
 - 3D 오브젝트 임포트
 - 3D 애니메이션 및 리타게팅
 - 애니메이터 상태머신
 - 애니메이션 블랜딩
 - 애니메이션 레이어
 - 파티클 이펙트
 - 콤보 시스템
 - 시네머신
 - 적 상태머신
 - 적 아이템 드랍
 - 인벤토리
   - 아이템 추가, 삭제, 이동, 분리, 사용, 장비
   - 마우스 오버, 클릭, 드래그 처리
   - 무기 및 방어구 장비 또는 해제
 - 미니맵
